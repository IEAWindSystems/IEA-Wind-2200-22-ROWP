# The IEA Wind 2200-22-MW Reference Offshore Wind Plants

--- This repo is under preparation. A detailed overview of the contents will be provided soon ---

This repository contains the data defining the IEA Wind 2200-22-MW Reference Offshore Wind Plants, encoded following the windIO ontology. Furthermore, the optimization routine used to derive the layouts, the windIO validation script, and example files to load the data and run FLORIS or PYWAKE are provided.

A WES paper describing data, metadata, file structure and method used to optimize the layouts is under preparation and will be linked here soon.
